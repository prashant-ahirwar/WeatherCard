## ✨ Features

- 🔍 Search weather by city
- 🌡 Real-time temperature and weather data
- ☁️ Dynamic weather illustrations using Cloudinary
- 🌙 Day / night weather states
- 📅 7-day weather forecast
- 💧 Humidity and feels-like temperature
- ⚡ Lightweight and fast
- 🎨 Modern card-based UI
- 🔐 Secure API key handling with backend server

## How it Works

User searches city
        ↓
Frontend sends request
        ↓
Node.js backend fetches WeatherAPI data
        ↓
Weather condition is analyzed
        ↓
Matching Cloudinary image is selected
        ↓
UI updates dynamically
